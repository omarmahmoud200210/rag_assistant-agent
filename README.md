# Agentic Personal Assistant 🤖

Welcome to my Agentic Personal Assistant project! 

## About Me & This Project
I am a self-taught full-stack developer, and this repository marks my very first dive into building a **RAG (Retrieval-Augmented Generation) Agent**. 

The primary purpose of this project isn't to build a production-ready enterprise application, but rather to **learn, experiment, and deeply explore the fascinating world of AI Agents**. By building this from scratch, I aim to understand the core mechanics of how Large Language Models (LLMs) connect with external systems to retrieve real-time knowledge and autonomously execute actions.

## Tech Stack 🛠️
This is a simple yet effective architecture built using modern tech:
- **Backend:** Node.js & Express.js
- **Frontend:** React with Tailwind CSS
- **AI/LLM Framework:** LangChain & LangGraph
- **Vector Database:** Pinecone
- **Inference/Models:** Groq (Llama 3) & Pinecone Embeddings / OpenAI

## Features 🚀
- **Document Ingestion:** Upload PDF files which are automatically chunked, embedded, and stored in a Pinecone vector database.
- **Agentic Search:** A conversational React interface where the central AI Agent autonomously decides when to query the vector database to answer your specific questions.
